# 🔗 url shortener

a full stack url shortener built with react and django rest framework. users can securely register, log in with jwt authentication, create short urls, edit or delete them and track the number of clicks for each link.

## ✨ features

* 🔐 jwt authentication
* ✂️ create short urls
* ✏️ edit existing urls
* 🗑️ delete urls
* 📋 copy short url to clipboard
* 📊 click analytics
* 📱 responsive ui
* 🔔 toast notifications
* 🐘 postgresql database
* ☁️ deployed on vercel and railway

## 🛠️ tech stack

### 🎨 frontend

* react
* react router
* axios
* react toastify
* react icons
* css

### ⚙️ backend

* django
* django rest framework
* simple jwt
* postgresql
* neon
* railway
